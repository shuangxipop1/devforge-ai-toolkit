# REST API Generator Template

## Input
- Database schema or OpenAPI spec
- Preferred language/framework
- Authentication requirements

## Output
Complete REST API with:
- CRUD endpoints
- Input validation
- Error handling
- API documentation
- Unit tests

## Flow
1. Analyze schema/spec
2. Design endpoint structure
3. Generate routes
4. Implement middleware
5. Add tests
6. Generate docs

## Frameworks Supported
- Node.js/Express
- Python/FastAPI
- Go/Gin
- Rust/Axum
