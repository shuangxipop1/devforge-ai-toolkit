# DevForge CLI Tools Bundle

**5 Production-Ready CLI Tools for Developer Productivity**

---

## Tools Included

### 1. git-log-analyzer
```
Analyzes your git history and shows:
- Most active files (maintenance burden)
- Code review time metrics
- Bus factor analysis
- Contributor statistics

Usage: glan --repo . --format summary
```

### 2. env-secrets-checker
```
Scans environment files for secrets:
- API keys, tokens, passwords
- Accidental commit detection
- Rotation reminders
- Supports .env, .env.local, .env.production

Usage: esc --scan .
```

### 3. dependency-health-check
```
Checks your package.json dependencies:
- Outdated packages with severity
- Security vulnerabilities
- Deprecated package warnings
- License compliance

Usage: dhc --package package.json
```

### 4. api-endpoint-docs
```
Auto-generates API documentation:
- Parses route files
- Extracts HTTP methods, paths, params
- Generates OpenAPI spec
- Creates markdown docs

Usage: aed --input ./routes --output ./docs
```

### 5. commit-msg-validator
```
Validates commit messages:
- Enforces conventional commits format
- Pre-commit hook setup
- CI integration ready

Usage: cmv --commit "feat: add new feature"
```
