# Debug Master Plugin

Advanced debugging with AI-powered root cause analysis.

## Features

- **Root cause analysis**: Traces errors to source
- **Stack trace parsing**: Intelligent stack trace analysis
- **Fix suggestions**: Provides context-aware fix recommendations
- **Historical patterns**: Uses patterns from past debugging sessions

## Usage

```
/debug-master "fix the race condition in auth module"
```

## Debugging Flow

1. **Capture**: Gather error context
2. **Analyze**: Trace root cause
3. **Suggest**: Propose fixes
4. **Verify**: Confirm fix works
5. **Document**: Record solution for future

## Compared to Manual Debugging

| Aspect | Manual | Debug Master |
|--------|--------|--------------|
| Speed | Minutes to hours | Seconds |
| Accuracy | Variable | High |
| Root cause | Often missed | Always traced |
| Learning | None | Persistent |
