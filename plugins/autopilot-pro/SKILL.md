# Autopilot Pro Plugin

Enhanced version of standard Autopilot with adaptive learning capabilities.

## Features

- **Self-correction**: Learns from execution patterns
- **Context preservation**: Maintains full context across iterations
- **Smart retry**: Backs off on failing strategies
- **Progress snapshots**: Saves state for recovery

## Usage

```
/autopilot-pro "build a complete REST API"
```

## Compared to Standard Autopilot

| Feature | Autopilot | Autopilot Pro |
|---------|----------|---------------|
| Self-correction | ✗ | ✓ |
| Context preservation | Limited | Full |
| Smart retry | Basic | Adaptive |
| Progress snapshots | ✗ | ✓ |
| Price | Free | $49 add-on |
