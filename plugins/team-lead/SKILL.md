# Team Lead Plugin

Advanced multi-agent coordination for complex projects.

## Features

- **Role assignment**: Dynamically assigns roles to agents
- **Task dependencies**: Manages complex dependency graphs
- **Progress tracking**: Real-time dashboard of agent progress
- **Conflict resolution**: Handles agent disagreements automatically

## Usage

```
/team-lead "build complete e-commerce platform"
```

## Agent Roles

- **Architect**: System design
- **Executor**: Implementation
- **Critic**: Quality assurance
- **Test Engineer**: Testing
- **Debug Master**: Issue resolution
- **Planner**: Task orchestration
