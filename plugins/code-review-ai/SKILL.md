# Code Review AI Plugin

Automated code review with deep security and performance analysis.

## Features

- **Security scan**: OWASP Top 10 vulnerability detection
- **Performance analysis**: Identifies N+1 queries, memory leaks
- **Style enforcement**: Configurable code style rules
- **PR summary**: Auto-generates PR descriptions

## Usage

```
/code-review "review PR #123"
```

## Check Categories

- Security vulnerabilities
- Performance anti-patterns
- Code style violations
- Best practice deviations
- Test coverage gaps

## Output

- Inline comments on issues
- Summary report
- Suggested fixes
- Priority rankings
